#### 时间复杂度

估算程序指令的执行次数

#### 空间复杂度

估算所需占用的存储空间

#### 大O表示法

一般用大O表示法来描述复杂度，它表示的是数据规模n对应的复杂度。

它一般有以下规则：

##### 忽略常数、系数、低阶

- 9 >> O(1)
- 2n + 3 >> O(n) 
- n^2 + 2n + 6 >> O(n^2)
- 4n^3 + 3n^2 + 22n + 100 >> O(n^3)
- lg(n) + n >> O(n)
- 3n + 2log(3)(n) + 15 >> O(nlgn)
- 2^n >> O(2^n)

##### 对数阶的细节

对数阶一般忽略底数

比如 log(2)(n) = log(2)(10) * log(10)(n)，而log(2)(10)则可以看成是常数项，**所以log(x)(n)统称为lg(n)**。

#### 常见的复杂度排序

O(1) < O(lgn) < O(n) < O(nlgn) < O(n^2) < O(n^3) < O(2^n) < O(n!) < O(n^n)
